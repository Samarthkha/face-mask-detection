# face-mask-detection
This project focuses on building a real-time Face Mask Detection System using computer vision techniques. The goal of the project is to automatically detect whether a person is wearing a face mask or not through a live webcam feed.

The system uses OpenCV for face detection and processes each frame captured from the webcam. Detected faces are analyzed and classified into two categories: “Mask” and “No Mask”. Based on the classification, the system displays bounding boxes with appropriate labels in real time.

The motivation behind this project comes from the need for automated monitoring systems in public places, especially during situations like the COVID-19 pandemic where mask compliance is important. Manual monitoring can be inefficient and error-prone, so this project demonstrates how computer vision can be used as a practical solution.

The implementation focuses on building a complete real-time pipeline including face detection, preprocessing, and classification logic. Due to time constraints, a simplified approach was used for classification, but the system successfully demonstrates the core concept and functionality.

Overall, this project highlights how computer vision techniques can be applied to solve real-world problems efficiently and provides a foundation for further improvements such as integrating advanced deep learning models or deploying the system on surveillance devices.
